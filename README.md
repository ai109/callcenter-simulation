# Callcenter Simulation (SimPy + SimLuxJS)

This repo contains two implementations of the same call-center model:

- simpy/ (Python + SimPy)
- simluxjs/ (Node.js + SimLuxJS)

## SimPy

- cd simpy
- python -m venv .venv
- source .venv/bin/activate
- pip install -r requirements.txt
- python callcenter_simpy.py

Quick run:
cd simpy && python -m venv .venv && source .venv/bin/activate && pip install -r requirements.txt && python callcenter_simpy.py

## SimLuxJS

- cd simluxjs
- npm run start

Quick run:
cd ../simluxjs && npm run start
